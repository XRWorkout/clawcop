Self-Host Pitfall

You buy a Mac Mini, spend a weekend configuring it, then realize you need it available when you're away from home. Now you're learning about Tailscale, dynamic DNS, and port forwarding.

VPS Pitfall

You spin up a $20/mo droplet. It works — until you need persistent browser sessions, GPU access, or more than 2GB of RAM. The upgrade path is a minefield of provider lock-in.

Hosted Pitfall

You sign up for a managed service. Easy — but now you're paying $200/mo for something you could run for $40 if you just knew how. And your data lives on someone else's server.

The Hidden Cost

Whichever you pick wrong, migration is brutal. Data, configs, skill setups, messaging bridges — nothing transfers cleanly. The cost of getting it wrong isn't the hosting bill. It's starting over.
